功能模块：

1. 文件处理模块

   ​	svg文件保存，打开，预览，另存为，新建

2. 界面实现模块

   ​	主界面和辅助界面的搭建，时间显示，网格绘制

3. 绘画功能模块

   ​	直线，贝塞尔曲线，矩形，圆，图片，文本框

4. 绘画处理模块

   ​	清屏，放大，缩小，还原
